# Support Ticket System using component using react
## MERN



## Features

- generate new ticket
- reply
- close ticket
- validation





## Tech



- React Js
- Node Js
- Express Js
- MongoDB


## Installation

## client side
```sh
cd client
npm install
npm start
```
## server side
```sh
cd server
npm install
cd src 
node app.js
```
